# CrackDN Object Detection

## Overview

CrackDN Object Detection is a task-specific package derived from CrackDataset_DL_HY (CrackDN-related box-level data).

- Task: object_detection
- Images: 2438
- Annotations: 2747
- Classes: 4 (transverse, alligator, longitudinal, sealed)
- Annotation format: PASCAL VOC XML bounding boxes

This package uses the "Multiple classes" subset and normalizes label typo `transeverse` to `transverse`.

## Source Dataset

- Name: CrackDataset_DL_HY (box-level detection subset)
- Source URL: https://github.com/juhuyan/CrackDataset_DL_HY/tree/master
- Article DOI: 10.1016/j.autcon.2019.102946
- Article title: Detection of sealed and unsealed cracks with complex backgrounds using deep convolutional neural network
